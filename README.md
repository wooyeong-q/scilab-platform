# SciLab

교사가 만든 과학 학습 프로그램을 한곳에서 찾고 실행하는 공유 플랫폼입니다.

## 개발 기준

- 모바일 360px부터 태블릿과 PC까지 핵심 기능을 동일하게 제공합니다.
- 드래그 기반 프로그램은 터치 선택 대체 방식을 함께 제공합니다.
- 터치 대상은 최소 44×44px, 입력 글자는 모바일에서 최소 16px을 사용합니다.
- 세로 화면을 기본 지원하고, 필요한 실험에만 가로 화면을 권장합니다.
- 운영 배포 전 타입 검사, 프로덕션 빌드와 모바일 화면 검수를 통과해야 합니다.

상세 기준과 기기별 검수표는 [모바일·디자인 품질 기준](docs/mobile-design-standard.md)을 참고하세요.

## 로컬 확인

```bash
npm install
npm run check
```
